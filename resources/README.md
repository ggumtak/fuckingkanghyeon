# 연결 리스트 빈칸 퀴즈

Python 연결 리스트 코드의 빈칸을 채우며 학습하는 인터랙티브 퀴즈 앱

## 🚀 실행 방법

`quiz.html`을 브라우저에서 열기

## 📁 구조

```
html/
├── quiz.html                 ← 메인 페이지
└── linked_list_quiz/
    ├── styles.css            ← 다크 테마 CSS
    ├── quiz-app.js           ← 채점/복습 로직
    ├── blank-practice.html   ← 백지복습 모드
    ├── quiz-1~6.html         ← 각 회차 퀴즈
    └── data/                 ← 퀴즈 데이터
```

## ⌨️ 단축키

| 키 | 기능 |
|---|------|
| `Enter` | 개별 채점 |
| `Enter` x2 | 정답 보기 (틀린 경우) |
| `Ctrl+Enter` | 전체 채점 |

## 🎨 상태 표시

- 🟢 **초록**: 정답 (수정 불가)
- 🟡 **노랑**: 틀렸다가 고침 (수정 불가)
- 🔴 **빨강**: 오답 또는 정답 확인

## 🔄 복습 기능

- **🔴 다시**: 정답 본 문제만 다시 풀기
- **🟡 다시**: 고친 문제도 포함해서 다시 풀기

## ➕ 새 퀴즈 추가

`/add-quiz-round` 워크플로우 또는 "이 빈칸 코드 추가해줘" 라고 요청
